# EnderNoInventoryMove

Ein Plugin, das das Verschieben von Gegenständen im Inventar verhindert.

## Installation

1. Lade die neueste Version von [GitHub Releases](https://github.com/Endkind/EnderNoInventoryMove/releases) herunter oder direkt von [PaperMC Hangar](https://hangar.papermc.io/Endkind_Ender/EnderNoInventoryMove).
2. Platziere die JAR-Datei im `plugins`-Ordner deines PaperMC-Servers.

## Unterstützung

Bei Problemen oder Fragen kannst du gerne ein Issue auf der [GitHub-Seite](https://github.com/Endkind/EnderNoInventoryMove/issues) des Plugins erstellen.

Besuche auch gerne unser Minecraft-Netzwerk, EnderMC, unter [mc.endkind.net](https://mc.endkind.net).
